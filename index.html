pvlib: open-source PV modeling in python

Performant: pvlib is built on highly-optimized libraries like NumPy and SciPy.

Flexible: Each modeling step is fully exposed so that you can build your own customized performance models.

Scalable: Run simulations in bulk using python's powerful scripting capabilities.

Friends of pvlib

The open-source PV ecosystem is growing all the time. Here are some packages closely-related to pvlib:

pvlib.iotools

Weather datasets come in many shapes and sizes. Fetch and parse them with ease in a single unified interface.

pvanalytics

Data from real-world systems is messy. Clean up the mess using pvanalytics.

pvfactors

There's no two ways about it: bifacial PV is complicated. Model the ups and downs with pvfactors.
